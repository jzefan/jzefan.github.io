互联网+

142015年07月

最近一年多来，中国互联网领域里最热的词应该是”互联网+”了，在中国的IT老总们不停地推销和各种自媒体/传统媒体不停地推波助澜下，在官方大力宣传下，”互联网+”俨然成了万能的催化剂，成了大众创业、万众创新的“新引擎”。那么“互联网+”到底是什么呢？

我的理解是：它仅仅是个商业词语而已，一个商业故事概念，或许是借鉴一些商业产品里的”plus”（比如iPhone plus:-)，它仍然是把互联网应用到各行各业去的一个工具，它并没有带来任何技术上的突破，或者出现任何新的技术，目前看来，它是在中国这个语境下，一些人鼓吹出来并辅助以“互联网思维”进行包装的一个概念，而这又恰恰被官方在当前经济环境下拿来作为经济改革的救命稻草之一。

“互联网+” 是互联网思维的进一步实践成果，它代表一种先进的生产力，推动经济形态不断的发生演变。从而带动社会经济实体的生命力，为改革、发展、创新提供广阔的网络平台。
通俗来说，“互联网+”就是“互联网+各个传统行业”，但这并不是简单的两者相加，而是利用信息通信技术以及互联网平台，让互联网与传统行业进行深度融合，创造新的发展生态。
它代表一种新的社会形态，即充分发挥互联网在社会资源配置中的优化和集成作用，将互联网的创新成果深度融合于经济、社会各领域之中，提升全社会的创新力和生产力，形成更广泛的以互联网为基础设施和实现工具的经济发展新形态。
几十年来，“互联网+”已经改造及影响了多个行业，当前大众耳熟能详的电子商务、互联网金融、在线旅游、在线影视、在线房产等行业都是“互联网+”的杰作。

让我们来看一些搜索数据吧：

使用google去查找”internet plus”，得出594,000结果，翻看第一页和第10页，其中大部分是和China相关的，不相关部分和所谓的互联网加好像也不在一个意思。同样的查找”互联网加”，得出来的也基本是国内的新闻和一些专业网络写手写的什么是互联网加，互联网加是什么之类的吹鼓文章。这么多搜索结果中，并没有看到一篇相关互联网加的技术文档，没有看到来自学术界或工业界的相关论坛，会议等（英文世界里查出的是相关中国的，中文世界里则都是一些概念性的东西）。所以结论是：它来自中国，并仅仅活跃于中国，它能不能被世界接受，让我们拭目以待。

事实上，在上一次互联网热潮时，我们经常能听到一个名词：”[Web 2.0]“ (https://en.wikipedia.org/wiki/Web_2.0)，看看里面的定义，你是否发现和现在的**互联网+**有点类似。

Web 2.0 can be described in three parts:

  • Rich Internet application (RIA) — defines the experience brought from desktop to browser whether it is from a graphical point of view or usability point of view.
  • Web-oriented architecture (WOA) — is a key piece in Web 2.0, which defines how Web 2.0 applications expose their functionality so that other applications can leverage and integrate the functionality providing a set of much richer applications. Examples are feeds, RSS, Web Services, mash-ups.
  • Social Web — defines how Web 2.0 tends to interact much more with the end user and make the end-user an integral part.
