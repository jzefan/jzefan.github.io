鑫鸿图物流管理项目

012015年07月

介绍

本软件是为钢材物流公司开发的提单录入，配发，价格结算，数据统计和分析报表的项目；是一个从初始数据到回款的一个完全业务循环过程的实现，记录着每一个过程状态维护的历史数据，从而为物流公司提供数据一致性以及数据变化的趋势性分析报告，有助于公司领导者进一步分析和决策

技术特点

软件技术上：

  1. 前端：HTML, CSS, Javascript (JQuery, Bootstrap)
  2. 后端：Node.js + Express 4.0 + MongoDB

功能模块:

  1. 提单录入， 导入标准模板（和南钢提单文件接口）
  2. 提单修改，删除
  3. 运单配发，修改，删除；包括车运单，船运单
  4. 运单报表，查询，打印，导出不同格式文件(PDF, Excel, Text)
  5. 客户/代收代付结算，开票，回款以及车船结算
  6. 运单，提单综合查询
  7. 客户营业额，车船营业统计，图表显示
  8. 数据字典，用户管理，权限管理等辅助功能

部分界面

1. 提单录入

2. 创建新运单（配发货)

3. 客户代收代付结算，开票


4. 车船结算

5. 客户营业额统计图表

6. 车船营业额统计
